# 백엔드 앱 실행 명령어

테스트 진행
```shell
npm test
```

swagger doc파일 빌드
```shell
npm run api-docs
```

development 환경에서 실행
```shell
npm run start:dev
```
production 환경에서 컴파일 후 실행
```shell
npm run start:prod
```
